The Herald Protocol facilitator supports the following networks and tokens. Network identifiers follow [CAIP-2](https://chainagnostic.org/CAIPs/caip-2).

</Note>

## Schemes

Both EVM schemes defined by x402 are registered on every supported network:

- **exact** — the buyer authorizes and pays the advertised price.
- **upto** — the buyer authorizes a maximum; the seller settles for the actual usage.

Use [`GET /supported`](https://facilitator.heraldprotocol.xyz/supported) to query the authoritative `(network, scheme)` list at runtime.

## Pricing in USD

When you use `ExactEvmScheme` from `@heraldprotocol/x402/server`, you can price endpoints in USD strings like `"$0.001"`. For Base and Arbitrum, the upstream `@x402/evm` library resolves the amount natively. For 0G Mainnet, Herald Protocol extends this with a parser that maps to Bridged USDC at the correct address — no hardcoded decimals or token addresses required.

```ts
import { ExactEvmScheme } from "@heraldprotocol/x402/server";

new x402ResourceServer(facilitator).register(
  "eip155:16661",
  new ExactEvmScheme()
);
```
